# Spoken English 微信小程序版本

这个目录是一套微信小程序原型，可用微信开发者工具打开 `miniprogram` 目录。

## 已实现

- 中文句子生成自然英文口语表达
- 场景切换：日常、工作、旅行、学习、历史文化、情绪支持
- 发音技巧提示：连读、略读、同化、浊化
- `wx.getRecorderManager()` 录音、停止、回放
- 多窗口英文聊天练习

## 还需要接入

- 自动语音评分：需要云函数或后端语音识别 API
- 文本朗读 TTS：可接腾讯云、微信同声传译插件或自建 TTS 服务
- 用户学习记录：可接云开发数据库

## 打开方式

1. 安装微信开发者工具
2. 导入项目，选择本目录 `miniprogram`
3. 没有 AppID 时可先使用测试号或游客模式
4. 真机调试时允许麦克风权限
